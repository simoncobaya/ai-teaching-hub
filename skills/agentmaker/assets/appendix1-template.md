## 填写指引

### 各类型说明

| 字段 | 填写要求 |
|------|----------|
| 智能体简介 | 200字以内，说清楚"是什么、解决什么问题、怎么做、有什么特点" |
| 开发平台 | 填写实际使用的平台（Coze/Dify等），平台不限 |
| 作品链接 | 必须是**可公开访问**的链接（Coze Bot链接需要发布为"网页链接"） |
| 报告视频链接 | 上传到云盘（百度网盘/阿里云盘）并生成分享链接，或直接上传到比赛提交系统 |
| 创新亮点 | 简短有力，突出与同类作品的最大差异 |

### 注意事项

- 所有信息请确保**真实准确**，特别是学校名称、选手姓名、身份证号
- 团队赛按**最高年级选手**的组别划分
- 文件命名：`附件1_组别-省份-城市-学校-选手姓名-作品名称`

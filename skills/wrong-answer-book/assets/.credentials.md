# 腾讯云 API 认证信息（模板）

> ⚠️ **使用方法**：复制到项目根目录并重命名为 `credentials.md`，然后填入真实密钥。

## 认证凭据

请将下方的占位符替换为你的腾讯云 API 密钥（可在 [腾讯云控制台 - API 密钥管理](https://console.cloud.tencent.com/cam/capi) 获取）：

| 字段 | 值 |
|------|-----|
| SecretId | `<YOUR_SECRET_ID>` |
| SecretKey | `<YOUR_SECRET_KEY>` |

## 使用说明

- 复制本模板文件为 `credentials.md` 并填入真实密钥
- 编辑 `credentials.md`，填入真实的 SecretId 和 SecretKey
- `credentials.md` 已被 `.gitignore` 忽略，不会提交到仓库
- 此凭据用于调用腾讯云 OCR 服务（`SubmitQuestionMarkAgentJob` / `DescribeQuestionMarkAgentJob`）
- Skill 执行时会检查 `credentials.md` 是否存在且已配置，未配置则提示用户先完成

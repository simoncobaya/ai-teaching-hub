# 🤖 跟着 AI 搭一个「魔法错题本」！

> 不需要会写代码，像和朋友聊天一样，一步一步让 AI 帮你做出一个超酷的工具！

---

## 怎么用这份指南？

下面每个彩色框里都是一句话，你**复制粘贴**到 CodeBuddy 对话框里就行。

每次 AI 回复后，再粘贴下一句。就像这样：

```
你 → 说一句
AI → 回复一堆
你 → 说下一句
AI → 再回复
...
```

**不要一次全发过去！** 一步一步来，AI 会更认真。

---

## 🎬 第一站：开场！告诉 AI 你想要什么

> 💡 **复制这句发给 AI：**

```
嗨！我想做一个"魔法错题本"工具。就是拍一张考试卷子的照片，它能自动找出我做错的题目，帮我整理成一个错题本。你能帮我做成一个 skill 吗？
```

> 📌 这是最重要的第一步！AI 会问你一些问题，先聊清楚再动手。

<details>
<summary>👀 AI 可能会怎么回复？</summary>

AI 可能会告诉你几种方案（比如直接用 AI 看图片、用 OCR 工具、或者做 skill），你可以告诉它你想做 skill，方便以后反复使用。
</details>

---

## 📸 第二站：告诉 AI 你怎么识别卷子

> 💡 **复制这句发给 AI：**

```
我用的是腾讯云的 SubmitQuestionMarkAgentJob 接口来识别卷子，这个 API 可以自动批改试卷。你能根据这个接口来设计 skill 吗？
```

> 📌 AI 需要知道你怎么"看懂"卷子。记得提前准备好你的腾讯云 SecretId 和 SecretKey。

<details>
<summary>🔑 需要准备什么？</summary>

去 [腾讯云控制台](https://console.cloud.tencent.com/cam/capi) 获取 API 密钥（SecretId 和 SecretKey）。放心，AI 会帮你把密钥写在配置文件里，不会被别人看到。
</details>

---

## 📝 第三站：让错题库更好用

> 💡 **复制这句发给 AI：**

```
生成的错题要存成好看的 Markdown 文件，每道错题要写清楚：这是什么大题、原题是什么、我的答案是什么、正确答案是什么、为什么错了。还要能查「数学有多少错题」这样的统计。
```

> 📌 这样你以后打开错题文件就像看一本排版好的书，还能随时问 AI 你的学习进度。

<details>
<summary>📊 Markdown 长什么样？</summary>

就像一本有标题、有表格的电子书。AI 会帮你设计一个好看的模板，里面有试卷名称、日期、错题列表、错误原因分析等等。
</details>

---

## 🔍 第四站：能查看、能删除

> 💡 **复制这句发给 AI：**

```
我还想能做到这些事：
1. 说「看看错题库」能列出所有试卷
2. 说「看 1.2」能看具体某道错题的详情
3. 说「掌握了 1.2」能标记这道题已经会了
4. 还能删除整张卷子或者单道错题
```

> 📌 这样你平时复习的时候就能随时翻看错题，掌握了就划掉，特别有成就感！

---

## 🖥️ 第五站（选修）：做一个命令行工具

> ⚠️ 这一站可以跳过，直接去第六站做网页版更有意思！

> 💡 **如果感兴趣，复制这句：**

```
帮我把这些功能做成一个命令行工具，输入命令就能操作。比如 python wrong_answer_book.py upload 上传卷子，python wrong_answer_book.py list 查看错题。
```

---

## 🌐 第六站：做一个漂亮的网页界面！

> 💡 **复制这句发给 AI：**

```
在浏览器里用更酷！帮我加一个网页界面吧。我想要：
- 暖色调的卡通风格，大圆角卡片
- 彩色统计数字（几张卷子、几道错题、掌握率）
- 一个大大的「上传试卷」按钮
- 点击卡片能看到每道错题的详情
- 每道题旁边有「掌握了」按钮
- 上传卷子的时候有好玩的加载动画
```

> 📌 这是最有趣的一步！你会在浏览器里看到一个真正的网站，颜色鲜艳，按钮超大，像在玩游戏一样管理你的错题。

<details>
<summary>💻 怎么打开网页？</summary>

AI 做好后，在终端输入：
```
python web_app.py
```
然后在浏览器打开 `http://127.0.0.1:5000` 就能看到了！
</details>

## 💡 小贴士

| 问题 | 答案 |
|------|------|
| 卡住了怎么办？ | 说「继续」或「刚才卡住了，请继续」 |
| AI 做错了？ | 说「不对，我要的是……」告诉它哪里不对 |
| 想重新来？ | 说「重新开始，我要做的是……」 |
| 密钥在哪？ | 腾讯云控制台 → API 密钥管理 |
| 每次会话太长？ | 可以分多次聊，AI 会记住之前聊过什么 |

# 🎮 Playground — 把想法变成可以玩的交互工具

> Anthropic 官方 Skill，市场安装量 **5.7 万**。让 Claude 生成可拖拽、可探索的交互式 HTML 工具。

---

## 🤔 它解决了什么问题？

你有没有遇到过这种情况——

> 想让 AI 调整一个设计，但说不清楚："阴影再大一点……不对太大了……圆角稍微圆一点……"

用文字描述视觉细节很费劲。**Playground 就是来解决这个问题的**——它把一个想法变成一个**可以拖滑块、实时预览、一键复制 prompt**的交互工具。

```
你说："帮我做一个调按钮样式的工具"
  ↓
Claude 生成一个 HTML 文件
  ↓
你在浏览器打开 → 拖滑块调颜色、圆角、阴影 → 实时看到效果
  ↓
调满意了 → 点复制 → 把 prompt 粘贴回 Claude → "按这个参数帮我改"
```

## 🔧 怎么用

### 基本流程

1. **提出需求**：对 Claude 说你想探索什么话题
2. **Claude 选模板**：从 6 种模板中选最合适的
3. **生成 HTML**：单个文件，无需外部依赖
4. **浏览器打开**：Claude 运行 `open xxx.html`
5. **拖拽探索**：调参数、看预览
6. **复制 prompt**：底部自动生成自然语言描述，一键复制回 Claude

### 必须满足的标准

- **单文件**：CSS 和 JS 全部内联，不依赖 CDN
- **即时预览**：每次拖控件立即更新，不需要点"应用"按钮
- **自然语言 prompt**：不说"borderRadius=16"，而是"圆角 16px，有明显的阴影"
- **复制按钮**：一键复制，有"已复制！"反馈
- **默认值 + 预设**：打开就能看，3-5 个命名预设一键切换
- **暗色主题**：系统字体做 UI，等宽字体做代码

## 📋 六种模板

Playground 有 6 种内置模板，Claude 会根据你的需求自动选择。**如果都不完全匹配，就选最接近的那个模板，改一改来适配**。

### 1. 🎨 设计探索（Design Playground）

**用途**：调整视觉设计参数——组件样式、布局、间距、颜色、字体。

**示例**：按钮样式、卡片布局、阴影系统、配色方案。

```
"帮我做一个调按钮样式的 playground，能调圆角、颜色、阴影、大小"
"做一个对比不同字体搭配的工具"
```

---

### 2. 📊 数据查询（Data Explorer）

**用途**：构建 SQL、API 调用、正则表达式等查询语句。

**示例**：SQL 查询构造器、API 参数调试器、正则表达式测试器。

```
"帮我做一个 SQL 查询构造器，能选表、加条件、排序"
"做一个正则表达式测试工具"
```

---

### 3. 🗺️ 概念图（Concept Map）

**用途**：学习和探索——概念图、知识图谱、学习路径规划。

**示例**：知识点关系图、学习路线图、主题探索器。

```
"帮我做一个编程概念图，展示 HTML/CSS/JS 之间的关系"
"帮我规划一个学 Python 的路线图"
```

---

### 4. 📝 文档审查（Document Critique）

**用途**：审查文档——逐条建议，支持批准/拒绝/评论。

**示例**：设计文档审查、技术方案评审、README 审查。

```
"帮我审查这份设计文档，做成可以逐条批准或拒绝的工具"
```

---

### 5. 🔍 代码审查（Diff Review）

**用途**：审查代码变更——逐行查看 diff，逐条评论。

**示例**：Git diff 审查、PR 审查、commit 审查。

```
"帮我把这次的 git diff 做成可以逐行审查的工具"
```

---

### 6. 🏗️ 代码架构图（Code Map）

**用途**：可视化代码架构——组件关系、数据流、分层图。

**示例**：项目结构图、组件依赖图、数据流向图。

```
"帮我画一张这个项目的组件依赖关系图"
```

## 🎮 适合学生吗？

**非常适合！** ⭐⭐⭐⭐⭐

| 原因 | 说明 |
|------|------|
| 🎮 好玩 | 拖滑块、点按钮、实时看到变化——像玩游戏 |
| 👀 直观 | 不用学代码就能理解参数和效果的关系 |
| 🔄 互动 | 探索→调整→复制→粘贴回 Claude，形成闭环 |
| 🏗️ 创造感 | "我做了个工具"的成就感 |

**最适合学生的模板**：设计探索（调颜色/字体/布局）和概念图（可视化知识点）。数据查询和代码审查模板对初学者较难。

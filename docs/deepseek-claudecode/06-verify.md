## 🧩 给你的 AI 装上"装备" — Plugin 入门

### 你已经能用 AI 写代码了，但……

你有没有发现两个问题：

1. **AI 做的网页都长一个样**——白底、紫色按钮、圆角卡片，看多了就觉得"又是这种风格"
2. **你说半天 AI 还是理解错**——"我想要活泼一点的配色" → AI 给了你荧光绿配粉红 😅

**Plugin 就是用来解决这些问题的。** 它就像给手机装 APP——装完之后，手机就多了新功能。Plugin 也是：装完之后，AI 就多了新本领。

```
你安装了 Plugin → AI 获得了新能力 → 你能做更多更酷的事！
```

### 📥 怎么安装 Plugin？

安装分两步。**下面这些命令很长，不用自己打——直接复制粘贴就行！** 用鼠标选中 → `Ctrl+C` 复制 → 回到 Claude Code → `Ctrl+V` 粘贴 → 回车。

**第一步：添加官方市场**

```
/plugin marketplace add anthropics/claude-plugins-official
```

**第二步：安装你想要的插件**

```
/plugin install <插件名>@claude-plugins-official
```

安装时 Claude Code 会问你装在哪里。用 `↓` 键移到第三个选项（写着 `local scope` 的那个），按回车确认：

```
 > Install for you (user scope)
    Install for all collaborators on this repository (project scope)
    Install for you, in this repo only (local scope)
```

- 👤 **User**（用户）→ 你所有项目都能用
- 👥 **Project**（项目）→ 项目里所有人都能用（会提交到 git）
- 🏠 **Local**（本地，推荐）→ 仅你、仅此项目，不提交到 git

> 💡 **推荐选择 Local（🏠 本地）**。不提交到 git，最适合个人项目。

#### 1. 🎨 frontend-design — 让 AI 做出不像"AI 做的"网页

| 项目 | 内容 |
|------|------|
| 安装量 | **94.8 万**（市场第一 🏆） |
| 来源 | Anthropic 官方 |
| 安装 | `/plugin install frontend-design@claude-plugins-official` |

**它解决什么问题？**

你有没有注意过——AI 生成的网页界面，看起来都差不多？

```
❌ 没有这个 Plugin：奶油色背景 + Inter 字体 + 紫色渐变 + 圆角卡片
                    → "AI 味"十足
✅ 有了这个 Plugin：独特配色 + 个性字体 + 大胆布局
                    → 像专业设计师做的
```

**它怎么做到的？**

这个 Plugin 教 Claude 遵守 6 个设计原则：

| 原则 | 什么意思 | 对学生来说 |
|------|---------|-----------|
| 🌱 从主题出发 | 先确定"这是什么东西、给谁看" | 做宠物商店 vs 太空游戏官网，设计应该完全不同 |
| 🦸 首屏即论点 | 第一眼决定用户要不要继续看 | 把最吸引人的东西放在打开页面的第一屏 |
| 🔤 排版承载个性 | 标题有趣吸睛，正文舒服耐读 | 别全用默认字体！标题和正文要有区分 |
| 📐 结构即信息 | 不要为了好看乱加装饰 | 每一条线、每一个编号都要有理由——问自己"为什么这里要有？" |
| 🎬 动效要有意地用 | 动画像盐，一点点提味就好 | 一个精彩的动画比十个零散的动画好 |
| 📏 复杂度匹配愿景 | 做复杂就做足细节，做简单就在间距对齐下功夫 | 不要半途而废——选了一个方向就要执行到位 |

#### 2. 🦸 superpowers — 让 AI 养成"先想清楚再动手"的习惯

| 项目 | 内容 |
|------|------|
| 安装量 | **82 万** |
| 开源 Star | **204K+**（GitHub 开源项目） |
| 安装 | `/plugin install superpowers@claude-plugins-official` |

**它解决什么问题？**

```
❌ 没有这个 Plugin：
你："帮我做一个XX"
AI：好的，开始写代码……（写了 200 行，方向全错了）
你：不对不对！

✅ 有了这个 Plugin：
你："帮我做一个XX"
AI：我先问你几个问题搞清楚需求……
   （一次只问一个，不轰炸你）
   聊清楚了 → 写出计划 → 你点头 → 再动手写代码
```

**完整的 7 步工作流：**

```
Step 1 🧠 头脑风暴    AI 先提问搞清楚，绝对不急着写代码
Step 2 📝 写计划      AI 把任务拆成小步骤（每步 2-5 分钟）
Step 3 📦 隔离工作区   创建 git 分支做实验（不影响已有代码）
Step 4 🤖 子Agent执行  派出"小弟"一个个完成任务
Step 5 🧪 TDD 测试先行 写代码前先写测试——"红了→绿了→整理"
Step 6 🔍 代码审查     AI 自己审查自己的代码
Step 7 🏁 收尾         合并代码、清理工作区、完美收工
```

**对学生来说，最有用的 2 个：**

- 🧠 `/brainstorming` — AI 像老师一样引导你："游戏有几个玩家？""要记录最高分吗？" 一次只问一个，你的回答决定了下一步

- 📝 `/writing-plans` — AI 写出详细的步骤清单："1. 创建 HTML 文件 2. 写游戏逻辑 3. 加样式 4. 测试"——你确认了它才动手

> ⚠️ Step 3-7 需要 git 知识。初学者先只用 `/brainstorming` 和 `/writing-plans`，其他的学完 git 再回来探索！

#### 3. 🎮 playground — 把想法变成"可以拖拽"的交互工具

| 项目 | 内容 |
|------|------|
| 安装量 | **5.7 万** |
| 来源 | Anthropic 官方 |
| 安装 | `/plugin install playground@claude-plugins-official` |

**它解决什么问题？**

想让 AI 调整设计，但说不清楚：

> "阴影再大一点……不对太大了……圆角稍微圆一点……颜色也换一下试试……"

用文字描述视觉细节 = 用筷子喝汤 🥢 ——费劲又说不清！

**Playground 的做法：**

```
你说："帮我做一个调按钮样式的工具"
  ↓
Claude 生成一个 HTML 文件
  ↓
浏览器打开 → 你拖滑块调颜色、圆角、阴影 → 实时看到效果 ✨
  ↓
调满意了 → 点"复制"按钮
  ↓
把 prompt 粘贴回 Claude → "按这个参数帮我改"
```


**6 种模板：**

| 模板 | 用来做什么 |
|------|-----------|
| 🎨 设计调参 | 调颜色、圆角、阴影、间距，实时预览 |
| 📊 数据探索 | 构建 SQL 查询、API 请求 |
| 🗺️ 概念图 | 探索知识结构、概念关系 |
| 📄 文档审查 | 逐条批注修改建议（批准/拒绝/评论） |
| 🔍 代码审查 | 逐行审查代码改动 |
| 🏗️ 代码架构图 | 查看组件关系、数据流 |

**试试看！**

```
帮我做一个 playground，用来调卡片的设计——
可以调圆角、阴影大小、背景颜色、边框粗细。
```

浏览器打开后，像玩游戏一样拖滑块就能调出你喜欢的设计！

### 🔧 Plugin 管理小贴士

安装了 Plugin 之后，你可能需要：

| 操作 | 命令 |
|------|------|
| 查看已安装 | `/plugin list` |
| 查看可用 Skill | `/skills` |
| 禁用某个 Plugin | `/plugin disable <插件名>` |

## 🌟 进阶挑战

游戏能玩了？太棒了！试试让 AI 帮你添加更多功能：

### 挑战 1：添加难度选择

```
请给猜数字游戏添加难度选择：
- 简单：1-50，10 次机会
- 普通：1-100，7 次机会
- 困难：1-500，8 次机会

```

### 挑战 2：记录最高分

```
请添加一个功能：记录玩家最少用了几次猜对，
并在游戏开始时显示历史最佳成绩。
把记录存到一个文件里，即使关了游戏也能保留。
```

### 挑战 3：做成网页版

```
请把这个猜数字游戏改成网页版，用 HTML + CSS + JavaScript。
要好看一点，有不同的颜色和动画效果。
（提示：先装 frontend-design 插件再试！）
```

### 挑战 4：添加提示功能

```
请添加一个提示功能：当玩家输入"h"时，
给一个提示（比如告诉玩家数字是奇数还是偶数）。
每局只能用 2 次提示。
```

### 挑战 5：用 Superpowers 重新做一次 🦸

**操作顺序：**

1. 先装 superpowers（如果还没装）：
   ```
   /plugin install superpowers@claude-plugins-official
   ```
   （用 `↓` 选第三个 Local，回车）

2. 装好后，对它说：
   ```
   我想做一个全新的项目——一个"单词学习卡片"网页应用。
   请先别急着写代码，先问清楚我的需求。
   ```

看看有了 brainstorming + writing-plans 之后，AI 和你互动的方式有什么不同！

### 挑战 6：做一个 Playground 🎮

```
先装 playground 插件，然后说：
"帮我做一个 playground，用来尝试不同的渐变背景——
可以调起始颜色、结束颜色、渐变角度，实时预览。"
```

浏览器里打开，拖一拖，然后复制 prompt 让 AI 帮你生成真正的渐变背景！

做完猜数字游戏还不过瘾？这里有更多项目想法，按难度排列：

### 🌱 入门级（每个约 15-30 分钟）

| 项目 | 描述 | 学什么 |

| 项目 | 描述 | 学什么 |

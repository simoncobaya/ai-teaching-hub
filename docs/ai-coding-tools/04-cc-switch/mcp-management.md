# 📦 MCP 服务器管理

> MCP 让 AI 工具获得超能力！学会用 CC Switch 管理 MCP 服务器。

---

## 🤔 什么是 MCP？

**MCP**（Model Context Protocol）是一种让 AI 工具连接外部服务的方式。

你可以把它想象成给 AI **装插件**：

| MCP 服务器 | 给 AI 什么能力 |
|-----------|--------------|
| 🔍 **网络搜索** | 让 AI 能搜索互联网上的信息 |
| 🌐 **网页读取** | 让 AI 能读取网页内容 |
| 👁️ **视觉理解** | 让 AI 能看懂图片 |
| 📂 **GitHub** | 让 AI 能访问 GitHub 仓库 |
| 🗄️ **数据库** | 让 AI 能读写数据库 |

---

## 🚀 在 CC Switch 中管理 MCP

CC Switch 的 MCP 管理功能在 **Extensions（扩展）** 区域。

### Step 1：打开 MCP 管理

1. 启动 CC Switch
2. 在左侧导航栏找到 **「Extensions」** 或 **「扩展」**
3. 选择 **「MCP Servers」**

---

### Step 2：了解 MCP 的绑定方式

CC Switch 中的 MCP 服务器是 **按工具绑定** 的——也就是说，你需要为每个工具单独配置要使用哪些 MCP。

| 工具 | 支持 MCP？ | 说明 |
|------|-----------|------|
| 🤖 **Claude Code** | ✅ 支持 | 完整的 MCP 支持 |
| 📟 **Codex** | ✅ 支持 | 支持 MCP 服务器 |
| 🔍 **Gemini CLI** | ✅ 支持 | 支持 MCP 服务器 |
| 🆓 **OpenCode** | ✅ 支持 | 支持 MCP 服务器 |
| 🦀 **OpenClaw** | ❌ 不支持 | OpenClaw 目前不支持 MCP 管理 |

> 💡 也就是说，你在 Claude Code 中配置的 MCP，不会自动在 Codex 中生效。需要分别配置。

---

### Step 3：添加 MCP 服务器

你可以通过以下方式添加 MCP 服务器：

**方式一：导入已有配置（推荐）**

如果你已经在 Claude Code 或其他工具中手动配置过 MCP，CC Switch 可以直接导入：

1. 在 MCP 管理页面，选择目标工具（如 Claude Code）
2. 点击 **「导入已有配置」**
3. CC Switch 会自动发现该工具已有的 MCP 配置并导入

> 💡 这是最方便的方式！不用重新配置，直接复用已有设置。

**方式二：手动添加**

1. 在 MCP 管理页面，点击 **「添加 MCP」**
2. 填写 MCP 服务器的信息：
   - **名称**：给 MCP 起个名字（如"网络搜索"）
   - **传输类型**：选择 `stdio`、`http` 或 `SSE`
   - **配置信息**：根据 MCP 的要求填写
3. 选择要绑定到哪个工具
4. 保存

---

### Step 4：启用和管理 MCP

添加 MCP 服务器后：

1. 在 MCP 列表中找到你添加的服务器
2. 点击 **开关** 启用或禁用
3. 可以随时编辑或删除 MCP 服务器

> 💡 修改 MCP 配置后，需要**重启对应的工具**才能生效。

## 🌟 常用 MCP 推荐

### 1. 🔍 网络搜索 MCP

让 AI 能搜索互联网，获取最新信息。

**适合场景**：
- 查找最新的技术文档
- 搜索解决方案
- 获取实时信息

### 2. 🌐 网页读取 MCP

让 AI 能读取网页内容。

**适合场景**：
- 阅读在线文档
- 获取网页上的代码示例
- 分析网页结构

### 3. 👁️ 视觉理解 MCP

让 AI 能看懂图片。

**适合场景**：
- 看懂 UI 设计图并生成代码
- 分析截图中的错误信息
- 理解流程图

### 4. 📂 GitHub MCP

让 AI 能访问 GitHub 仓库。

**适合场景**：
- 查看开源项目代码
- 分析仓库结构
- 学习别人的代码

## ❓ 常见问题

### Q：MCP 需要额外付费吗？

大部分 MCP **免费使用**。但某些需要调用付费 API 的 MCP（如视觉理解）会消耗你的 API 额度。

### Q：启用了 MCP 但不生效？

1. **重启对应的工具**（如退出 Claude Code 再重新打开）
2. 在 Claude Code 中用 `/mcp` 命令查看已连接的 MCP
3. 确认 MCP 配置的传输类型和参数是否正确

### Q：可以添加自己的 MCP 吗？

可以！在 CC Switch 的 MCP 管理页面，手动添加 MCP 服务器，填入服务器信息即可。

### Q：OpenClaw 为什么不支持 MCP？

OpenClaw 是一个多供应商网关工具，它的架构和其他工具不同，目前 CC Switch 还没有为它实现 MCP 管理功能。未来版本可能会支持。

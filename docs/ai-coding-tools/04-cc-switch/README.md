# 🔧 CC Switch 是什么？

> CC Switch 就像一个 **AI 工具的万能遥控器**，帮你一键管理所有 AI 编程工具的配置！

---

## 🤔 CC Switch 是什么？

CC Switch 是一个 **跨平台桌面应用**，专门用来管理 AI 编程工具的配置。

你有没有觉得：
- 😫 每个工具的配置方式都不一样？
- 😫 切换模型要手动改配置文件？
- 😫 API Key 散落在各处不好管理？

**CC Switch 解决了所有这些问题！**

---

## 🌟 CC Switch 的功能

### 1. 🔌 统一管理 API 供应商

在一个界面里管理所有 API 供应商：
- 🐋 DeepSeek
- 🧠 智谱 GLM
- 🤖 Anthropic Claude
- 📝 OpenAI GPT
- ...还有 50+ 其他供应商

### 2. ⚡ 一键切换

不需要手动编辑配置文件，**点一下按钮** 就能：
- 切换 AI 模型
- 切换 API 供应商
- 切换目标工具

### 3. 🔧 管理多个工具

CC Switch 支持这些 AI 工具：

| 工具 | 类型 |
|------|------|
| **Claude Code** | 终端编程助手 |
| **Codex CLI** | OpenAI 终端工具 |
| **OpenCode** | 开源编程工具 |
| **OpenClaw** | 开源编程工具 |
| **Gemini CLI** | Google 终端工具 |
| **Hermes** | 编程助手 |
| **Claude Desktop** | Claude 桌面版 |

### 4. 📦 MCP 服务器管理

MCP（Model Context Protocol）是一种让 AI 工具连接外部服务的方式。CC Switch 可以帮你管理 MCP 服务器，比如：
- 🔍 网络搜索
- 📁 文件管理
- 🗄️ 数据库连接

### 5. 🎯 Skills 管理

管理 Claude Code 的 Skills（技能包），一键安装和卸载。

---

## 📊 CC Switch vs 手动配置

| 操作 | 手动配置 | CC Switch |
|------|---------|-----------|
| 添加 API Key | 找文件、编辑 JSON | 粘贴 Key，点保存 ✅ |
| 切换模型 | 改配置文件、重启 | 点一下按钮 ✅ |
| 管理多个工具 | 每个工具分别配置 | 一个界面管理所有 ✅ |
| 添加新供应商 | 查文档、写配置 | 从预设中选择 ✅ |

## 🤔 智谱 GLM 是什么？

**智谱 AI**（Zhipu AI）是清华大学背景的 AI 公司，开发的 **GLM 系列模型** 在编程方面表现优秀。

### 🌟 为什么要用 GLM？

| 特点 | 说明 |
|------|------|
| 🇨🇳 **中文最强** | 国内 AI 模型中中文理解能力数一数二 |
| 💻 **编程能力强** | GLM-5.1 编程能力达到 Claude Opus 的 94.6% |
| 💰 **价格实惠** | Coding Plan 订阅制，不限量使用 |
| 🔌 **兼容性好** | 支持 Claude Code / Cursor / Cline / OpenCode 等 |
| 🎁 **专属 MCP** | 视觉理解、网络搜索、网页读取、GitHub 等专属 MCP |

## 📦 GLM 的模型

| 模型 | 特点 | 适合场景 |
|------|------|---------|
| **GLM-5.1** | 最新旗舰，最强推理 | 复杂编程任务 |

## 💰 Coding Plan（编程套餐）

智谱推出了 **Coding Plan**，订阅后可以不限量使用 GLM 模型编程。

> ⚠️ **重要**：智谱有 **国内版** 和 **国际版** 两个独立平台，**账号不互通**，定价也不同！

### 🇨🇳 国内版（bigmodel.cn）— 推荐国内用户

| 套餐 | 价格 | 适合 |
|------|------|------|
| **Lite** | ¥10/月 | 轻度使用 |
| **Pro** | ¥30/月 | 日常编程 |
| **Max** | ¥80/月 | 重度使用 |

### 🌍 国际版（z.ai）

| 套餐 | 月付 | 季付（-10%） | 年付（-20%） |
|------|------|-------------|-------------|
| **Lite** | $18/月 | $16.2/月 | - |
| **Pro** | $72/月 | $64.8/月 | - |
| **Max** | $160/月 | $144/月 | - |

> 💡 国内版价格是国际版的 **1/10 左右**，国内用户请务必使用国内版！

> 💡 Coding Plan 支持在 Claude Code、Cursor、Cline、OpenCode、Roo Code、Kilo Code、Crush、Goose 等工具中使用。

## 🔧 怎么用？

### 方式一：配置到 Claude Code（推荐）

智谱提供 **Anthropic 兼容端点**，可以直接在 Claude Code 中使用：

- API 端点：`https://open.bigmodel.cn/api/anthropic`
- 和 DeepSeek 配置方式类似

👉 **[详细配置教程](./configure-claudecode.md)**

### 方式二：配置到 Cursor

也可以在 Cursor IDE 中使用 GLM：

👉 **[配置 Cursor 使用 GLM](./configure-cursor.md)**

### 方式三：用自动化工具

智谱提供了自动化配置工具：

```bash
# 一键配置助手
npx @z_ai/coding-helper
```

### 专属 MCP 服务器

智谱 Coding Plan 用户可以使用独家 MCP 服务器：

👉 **[专属 MCP 服务器](./mcp-servers.md)**

# 🛠️ ZCode 编程工具简介

> ZCode 是智谱 AI 推出的编程工具，与 GLM 模型深度集成。

---

## 🤔 ZCode 是什么？

ZCode 是智谱 AI 基于 GLM 模型开发的编程辅助工具。它可以：

- 💻 智能代码补全
- 💬 AI 对话编程
- 🔄 代码重构和优化
- 🐛 Bug 检测和修复

---

## 🌟 ZCode 的特点

| 特点 | 说明 |
|------|------|
| 🧠 **GLM 原生** | 与 GLM 模型深度集成，发挥模型最大能力 |
| 🇨🇳 **中文优化** | 中文理解和生成能力特别强 |
| 🔌 **MCP 支持** | 支持智谱专属 MCP（视觉/搜索/网页/GitHub） |
| 📦 **Coding Plan** | 包含在 Coding Plan 订阅中 |

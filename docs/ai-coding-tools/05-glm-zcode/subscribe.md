## ⚠️ 国内版 vs 国际版

| 对比 | 🇨🇳 国内版 (bigmodel.cn) | 🌍 国际版 (z.ai) |
|------|-------------------------|-------------------|
| 注册方式 | 手机号 | 邮箱 / Google |
| 支付方式 | 支付宝 / 微信 | 信用卡 |
| 货币 | 人民币 (¥) | 美元 ($) |
| API 端点 | `open.bigmodel.cn` | `api.z.ai` |
| 价格 | 便宜（¥10起/月） | 较贵（$18起/月） |

> 💡 **国内用户请用国内版**，价格是国际版的 **1/10 左右**！

## 🚀 订阅步骤

### 🇨🇳 国内版订阅

#### Step 1：注册国内版账号

1. 打开智谱开放平台：**https://bigmodel.cn/**
2. 点击右上角的 **「注册/登录」**
3. 用手机号或邮箱注册

#### Step 2：进入 Coding Plan 页面

登录后，前往：**https://bigmodel.cn/glm-coding**

或点击菜单中的 **「编程套餐」**。

#### Step 3：选择套餐并订阅

1. 在套餐详情页，查看各个套餐的说明
2. 选择适合你的套餐
3. 点击 **「订阅」**
4. 完成支付（支付宝/微信）

#### Step 4：获取 API Key

订阅成功后：

1. 进入 **「个人编程套餐」** → **「套餐概览」**
2. 点击 **「新建 API Key」**
3. 给 Key 起个名字（如"我的编程 Key"）
4. 复制并保存好 API Key

> ⚠️ **团队套餐用户**：请到「团队编程套餐」→「我的套餐」获取 API Key。团队 Key 和个人 Key 不通用！

---

### 🌍 国际版订阅

#### Step 1：注册国际版账号

1. 打开 z.ai：**https://z.ai/subscribe**
2. 用邮箱或 Google 账号注册

#### Step 2：选择套餐

1. 选择 Lite / Pro / Max
2. 选择月付/季付/年付
3. 完成支付（信用卡）

#### Step 3：获取 API Key

1. 登录 z.ai 控制台
2. 创建 API Key
3. 保存好

## 🔄 使用 Coding Tool Helper（一键配置）

智谱提供了自动化配置工具，帮你一键把 GLM 配置到编程工具中：

```bash
npx @z_ai/coding-helper
```

运行后按照提示操作，它会自动帮你：
- 安装编程工具
- 配置 API Key
- 配置 MCP 服务器

> 💡 **推荐新手使用**，比手动配置简单很多！

## ❓ 常见问题

### Q：国内版和国际版账号可以互通吗？

**不可以！** 国内版（bigmodel.cn）和国际版（z.ai）是 **两个独立平台**：
- 账号不互通
- API Key 不通用
- 套餐和额度各自独立
- 需要分别注册

### Q：Coding Plan 和普通 API 有什么区别？

| | Coding Plan | 普通 API |
|--|------------|---------|
| 计费方式 | 月费（不限量） | 按 token 付费 |
| 适合 | 经常编程 | 偶尔使用 |
| 限制 | 仅限支持的编程工具 | 无限制 |

### Q：可以中途更换套餐吗？

可以！升级或降级都可以在套餐管理页面操作。

### Q：套餐到期后数据会丢失吗？

不会。你的项目代码都在本地电脑上，不受套餐影响。套餐到期只是不能再调用 GLM 模型。

# 🟣 Qoder

> Qoder 是阿里巴巴推出的 AI 编程平台，支持 IDE 和 CLI 双模式，有国内版和国际版。

---

## 🤔 Qoder 是什么？

Qoder 是阿里巴巴集团推出的 **新一代 AI 编程平台**。它提供：

- 🖥️ **Qoder IDE**：图形化开发环境（类似 Cursor），支持 Quest 自主编程模式
- 📟 **Qoder CLI**：终端命令行工具（类似 Claude Code），被称为"国内版 Claude Code 的最佳平替"
- 🧩 **JetBrains 插件**：在 JetBrains IDE 中使用
- 🇨🇳 **双版本**：国内版和国际版，价格差异大

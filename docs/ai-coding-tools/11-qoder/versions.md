# 🌍 国内版 vs 国际版

---

## 对比详情

| 对比项 | 🇨🇳 国内版（QoderCN） | 🌍 国际版 |
|--------|----------------------|-----------|
| **官网** | https://qoder.com.cn/ | https://qoder.com/ |
| **专业版价格** | **¥59/月** | **¥130/月** |
| **语言** | 中文 | 英文（多语言） |
| **服务区域** | 中国大陆 | 全球 |
| **网络要求** | 国内直连 | 国际网络 |
| **AI 模型** | 阿里云百炼平台 | 国际模型平台 |
| **注册方式** | 手机号 / 支付宝 | 邮箱 / Google |
| **合规认证** | 公安部三级等保（C3） | 国际标准 |
| **数据存储** | 国内 | 海外 |

---

## 产品形态（国内版和国际版都支持）

| 产品 | 说明 |
|------|------|
| 🖥️ **Qoder IDE** | 图形化开发环境，类似 Cursor |
| 📟 **Qoder CLI** | 终端命令行工具，类似 Claude Code |
| 🧩 **JetBrains 插件** | 在 JetBrains IDE 中使用 |

> 国内版和国际版都包含以上三种形态。

---

## 如何选择？

### 🇨🇳 选国内版，如果：
- 你在中国大陆使用
- 你希望价格更便宜（¥59 vs ¥130）
- 你需要中文体验
- 你有支付宝或国内手机号
- 你需要合规认证（三级等保）

### 🌍 选国际版，如果：
- 你在海外使用
- 你需要国际模型
- 你有 Google 账号或信用卡
- 你习惯英文界面

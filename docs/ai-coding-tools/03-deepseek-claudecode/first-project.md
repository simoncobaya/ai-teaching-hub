# 🎮 实践：让 AI 帮你写个猜数字游戏

> 配置完成后，让我们马上动手做一个有趣的小项目！体验 AI 编程的魅力！

---

## 🎯 目标

用 Claude Code + DeepSeek，从零开始做一个 **猜数字游戏**！

完成后你将拥有：
- ✅ 一个可以玩的 Python 小游戏
- ✅ 第一次体验"用 AI 写程序"的全过程
- ✅ 信心！你已经可以用 AI 来编程了！

---

## 🚀 开始动手

### Step 1：创建项目文件夹

打开终端，创建一个新的项目文件夹：

```bash
mkdir guess-game
cd guess-game
```

---

### Step 2：启动 Claude Code

```bash
claude
```

---

### Step 3：开始对话

确认你已经完成了 DeepSeek 的配置，直接开始对话就行！

---

### Step 4：告诉 AI 你想做什么

在 Claude Code 中输入：

```
帮我用 Python 写一个猜数字游戏：

1. 电脑随机选一个 1 到 100 之间的数字
2. 玩家输入猜测的数字
3. 如果猜大了，提示"太大了！"
4. 如果猜小了，提示"太小了！"
5. 如果猜对了，显示"恭喜你猜对了！"和猜了多少次
6. 最多猜 7 次，用完还没猜到就显示正确答案

请写一个完整的、可以直接运行的程序。
```

---

### Step 5：确认 AI 的操作

Claude Code 会告诉你它想做什么，比如：

```
📝 创建文件：guess_game.py
```

按 `y` 确认，或者按 `a` 允许所有后续操作。

---

### Step 6：运行游戏

等 Claude Code 写完代码后，输入：

```
运行这个游戏
```

Claude Code 会帮你运行：

```bash
python guess_game.py
```

---

### Step 7：玩游戏！🎮

现在你可以在终端里玩游戏了！

```
🎯 我想了一个 1-100 之间的数字，你来猜！
你有 7 次机会。

第 1 次猜测：50
📈 太大了！

第 2 次猜测：25
📉 太小了！

第 3 次猜测：37
📈 太大了！

第 4 次猜测：31
📉 太小了！

第 5 次猜测：34
🎉 恭喜你猜对了！
你用了 5 次猜对了数字 34！
```

## 🌟 进阶挑战

游戏能玩了？太棒了！试试让 AI 帮你添加更多功能：

### 挑战 1：添加难度选择

```
请给猜数字游戏添加难度选择：
- 简单：1-50，10 次机会
- 普通：1-100，7 次机会
- 困难：1-500，8 次机会

开始游戏时让玩家选择难度。
```

### 挑战 2：记录最高分

```
请添加一个功能：记录玩家最少用了几次猜对，
并在游戏开始时显示历史最佳成绩。
```

### 挑战 3：做成网页版

```
请把这个猜数字游戏改成网页版，用 HTML + CSS + JavaScript。
要好看一点，有不同的颜色和动画效果。
```

### 挑战 4：添加提示功能

```
请添加一个提示功能：当玩家输入"h"时，
给一个提示（比如告诉玩家数字是奇数还是偶数）。
每局只能用 2 次提示。
```

## 💡 你学到了什么？

通过这个项目，你体验了 AI 编程的完整流程：

```
描述需求 → AI 写代码 → 确认操作 → 运行测试 → 提出修改 → 不断改进
```

这就是 AI 编程的工作方式！你不需要记住所有语法，只要：
1. 📝 **清楚地描述**你想做什么
2. ✅ **检查** AI 写的代码对不对
3. 🔧 **提出修改**让程序更好
4. 🎉 **享受**创造的乐趣！

# 🎨 TRAE IDE

> TRAE 是字节跳动推出的 AI IDE，内置 Claude、GPT-4 等顶级 AI 模型！

---

## 🤔 TRAE 是什么？

TRAE 是字节跳动（就是做抖音/TikTok 的公司）开发的 **AI 代码编辑器**。

它最大的特点是：**免费体验** Claude、GPT-4 等顶级 AI 模型，付费可解锁更多功能！

### TRAE 的产品形态

| 产品 | 适合谁 | 说明 |
|------|--------|------|
| 🖥️ **TRAE IDE** | 个人用户（有免费版） | AI 代码编辑器，有 IDE 模式和 SOLO 模式 |
| 📟 **TRAE CLI** | 企业用户（付费） | 终端 Code Agent，仅企业版可用 |
| 🧩 **TRAE 插件** | VS Code 用户 | 在 VS Code 中使用 TRAE 的 AI 能力 |

> ⚠️ **注意**：**TRAE CLI 仅企业版可用**。个人用户只能使用 TRAE IDE。
>
> TRAE CLI 适用于：大规模重构、批量代码生成、代码迁移、测试补齐、CI/CD 集成等企业级场景。
>
> 如果你需要命令行编程工具，推荐使用 **Claude Code + DeepSeek** 或 **OpenCode**。

## 🌟 三种模式

### 📝 IDE 模式

就像普通的代码编辑器，但内置了 AI 助手。

适合：日常编程、代码编辑

### 🔧 Builder 模式

你和 AI 一起合作开发。你说需求，AI 帮你实现。

适合：新项目开发、功能开发

### 🤖 SOLO 模式

AI 自动驾驶模式！你只需要描述需求，AI 全自动帮你完成。

适合：快速原型、简单项目

> 💡 TRAE SOLO 现在还有独立的桌面端和手机端，可以在网页、桌面、手机上使用！

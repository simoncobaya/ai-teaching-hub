# 📥 下载安装 TRAE

---

## 下载

选择你需要的版本：

| 版本 | 下载地址 | 适合 |
|------|---------|------|
| 🌍 国际版 | https://www.trae.ai/download | 海外用户 |
| 🇨🇳 国内版 | https://www.trae.cn/ide/download | 国内用户 |

---

## 安装步骤

### Windows

1. 下载 `.exe` 安装包
2. 双击运行
3. 按安装向导操作
4. 完成！

### macOS

```bash
# Homebrew（推荐）
brew install --cask trae-ide

# 或手动下载 .dmg 安装
```

### Linux

```bash
# Ubuntu/Debian
sudo dpkg -i trae_xxx.deb

# Fedora/RHEL
sudo rpm -i trae_xxx.rpm
```

---

## 首次启动

1. 打开 TRAE
2. 注册/登录（支持 Google/GitHub 登录）
3. 选择模式：**IDE 模式**（推荐新手）
4. 开始使用！

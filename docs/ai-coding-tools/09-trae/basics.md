# 📖 TRAE 基础用法

---

## 三种模式

### 📝 IDE 模式（推荐新手）

和 VS Code 类似的编辑器体验，AI 随时待命：

- **Tab 补全**：写代码时自动建议
- **Chat**：和 AI 对话（`Ctrl + L`）
- **Inline Chat**：在代码中直接问 AI

### 🔧 Builder 模式

1. 切换到 Builder 模式
2. 描述你想做什么
3. AI 分析需求 → 制定计划 → 逐步实现
4. 你可以随时调整方向

### 🤖 SOLO 模式

1. 切换到 SOLO 模式
2. 描述完整的项目需求
3. AI 全自动帮你开发
4. 完成后你检查和测试

---

## 🎮 实战：用 SOLO 模式做一个小游戏

1. 切换到 **SOLO 模式**
2. 输入：
   ```
   用 HTML + CSS + JavaScript 做一个贪吃蛇游戏：
   - 方向键控制蛇的移动
   - 吃到食物蛇变长，得分增加
   - 撞到墙壁或自己就游戏结束
   - 界面要好看
   ```
3. 等 AI 自动帮你完成
4. 检查代码，运行测试

---

## 💡 使用技巧

| 技巧 | 说明 |
|------|------|
| 🎯 **描述要具体** | 越具体的描述，AI 做得越好 |
| 📋 **分步骤** | 复杂项目分步骤描述 |
| 🔍 **检查代码** | AI 写完要自己检查 |
| 📸 **截图反馈** | 用截图告诉 AI 哪里不对 |

| Chat | ✅ | ✅ |

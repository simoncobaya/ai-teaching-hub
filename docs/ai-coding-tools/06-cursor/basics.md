# 📖 Cursor 基础用法

> 学会 Cursor 的三大核心功能：Tab 补全、AI Chat、Composer！

---

## ⌨️ 功能一：Tab 智能补全

Cursor 最方便的功能就是 **AI 智能补全**。

### 怎么用？

1. 在编辑器中正常写代码
2. 当你停下来时，Cursor 会 **自动预测** 你接下来想写什么
3. 预测的内容会以 **灰色文字** 显示
4. 按 **Tab** 接受建议

### 技巧

| 操作 | 效果 |
|------|------|
| `Tab` | 接受全部建议 |
| `Ctrl + →` | 逐词接受 |
| `Esc` | 拒绝建议 |

---

## 💬 功能二：AI Chat（Ctrl+L）

在编辑器里直接和 AI 对话！

### 打开方式

- **Windows/Linux**：`Ctrl + L`
- **macOS**：`Cmd + L`

### 常见用法

#### 问问题
```
这段代码是什么意思？
```

#### 解释代码
```
选中一段代码 → 右键 → "Explain"（解释）
```

#### 找 Bug
```
这段代码运行报错了，帮我看看哪里有问题：
[粘贴错误信息]
```

#### 生成代码
```
帮我写一个 Python 函数，判断一个数是不是素数
```

---

## ✏️ 功能三：Composer（Ctrl+I）

让 AI **直接帮你修改代码**！

### 打开方式

- **Windows/Linux**：`Ctrl + I`
- **macOS**：`Cmd + I`

### 使用方法

1. 选中一段代码
2. 按 `Ctrl + I` 打开 Composer
3. 用自然语言描述你想怎么修改
4. AI 会生成修改方案
5. 点击 **Accept** 接受修改

### 示例

```
选中的代码 → Ctrl+I → "把这个函数改成支持列表输入"
```

Cursor 会直接帮你修改代码！

---

## 🤖 功能四：Agent 模式（@ 键）

使用 `@` 符号可以让 AI 访问更多信息：

| 符号 | 作用 |
|------|------|
| `@file` | 引用项目中的文件 |
| `@folder` | 引用整个文件夹 |
| `@web` | 搜索互联网 |
| `@docs` | 查找文档 |
| `@codebase` | 搜索整个代码库 |

### 示例

```
@file main.py 这个文件里的函数有什么问题？
```

```
@codebase 项目中哪里用到了数据库连接？
```

## 🎮 实战练习

### 练习 1：用 Chat 写代码

1. 按 `Ctrl + L` 打开 Chat
2. 输入：
   ```
   帮我写一个 Python 猜数字游戏
   ```
3. AI 会给出完整的代码
4. 点击 "Apply" 把代码应用到文件中

### 练习 2：用 Composer 改代码

1. 创建一个 Python 文件，写一个简单的函数
2. 选中函数代码
3. 按 `Ctrl + I`
4. 输入："加上错误处理"
5. 看看 AI 怎么帮你改

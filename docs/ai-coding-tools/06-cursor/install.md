# 📥 下载安装 Cursor

> Cursor 安装超简单，和安装普通软件一样！

---

## 🚀 安装步骤

### Step 1：下载 Cursor

打开 Cursor 下载页面：**https://cursor.com/download**

网站会自动检测你的操作系统，提供对应的安装包。

---

### Step 2：安装

#### Windows

1. 双击下载的 `.exe` 安装包
2. 按照安装向导操作
3. 安装完成！

#### macOS

1. 双击下载的 `.dmg` 文件
2. 把 Cursor 拖到 Applications 文件夹
3. 在启动台打开 Cursor

#### Linux

```bash
# Ubuntu/Debian
sudo dpkg -i cursor_xxx.deb

# AppImage
chmod +x cursor_xxx.AppImage
./cursor_xxx.AppImage
```

---

### Step 3：从 VS Code 迁移（可选）

如果你之前用 VS Code，Cursor 可以 **一键导入** 你所有的设置和插件：

1. 首次启动 Cursor
2. 会弹出导入提示
3. 选择"从 VS Code 导入"
4. 所有设置、主题、插件自动迁移！

> 💡 Cursor 和 VS Code 使用体验几乎一样，你可以无缝切换。

---

### Step 4：注册/登录

首次使用需要注册一个 Cursor 账号（可以用 Google 账号登录）。

---

## ✅ 验证

启动 Cursor 后，试试按 `Ctrl + L`（Windows/Linux）或 `Cmd + L`（macOS）打开 AI Chat。

如果能看到聊天界面，说明安装成功！🎉

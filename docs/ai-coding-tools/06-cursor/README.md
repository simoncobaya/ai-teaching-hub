# 🖥️ Cursor 是什么？

> Cursor 是目前最流行的 AI 代码编辑器，就像 VS Code 的超级升级版！

---

## 🤔 Cursor 是什么？

Cursor 是一个 **AI 原生的代码编辑器**，基于 VS Code 开发。它把 AI 能力深度融入到了编辑器的每个角落。

可以这样理解：
- 它长得像 VS Code（因为就是基于 VS Code 的）
- 但比 VS Code 多了很多 AI 功能
- 你可以用自然语言让 AI 帮你写代码、改代码

---

## 🌟 Cursor 的核心功能

### 1. ⌨️ 智能补全（Tab）

写代码时，Cursor 会 **预测你想写什么**，按 `Tab` 就能自动补全。

```
你输入：def hello
Cursor 预测：def hello_world():
                print("Hello, World!")
```

### 2. 💬 AI Chat（Ctrl+L）

在编辑器里直接和 AI 对话：
- 问问题
- 解释代码
- 让 AI 帮你找 bug

### 3. ✏️ Composer（Ctrl+I）

让 AI **直接帮你编辑代码**：
- 描述你想修改什么
- AI 直接改好文件
- 你只需要确认

### 4. 🤖 Agent 模式

让 AI **自主完成任务**：
- 分析整个项目
- 自动修改多个文件
- 运行测试

## 🆚 Cursor vs VS Code

| 特点 | Cursor | VS Code |
|------|--------|---------|
| AI 补全 | ✅ 内置 | 需要装插件 |
| AI 对话 | ✅ 内置 | 需要装插件 |
| AI 编辑 | ✅ Composer | ❌ |
| VS Code 插件 | ✅ 兼容 | ✅ |
| 价格 | 免费版/Pro $20/月 | 免费 |

> 💡 **Cursor 兼容所有 VS Code 插件**！你可以把 VS Code 的插件都搬到 Cursor 上。

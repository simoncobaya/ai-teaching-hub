# 🆓 OpenCode

> OpenCode 是一个 100% 免费开源的 AI 编程工具，支持各种 AI 模型，没有厂商锁定！

---

## 🤔 OpenCode 是什么？

OpenCode 是一个 **完全免费开源** 的终端编程工具。和 Claude Code 类似，但它最大的特点是 **不绑定任何 AI 厂商**。

### 🌟 为什么选 OpenCode？

| 特点 | 说明 |
|------|------|
| 🆓 **完全免费** | 开源，永远免费 |
| 🔓 **没有锁定** | 可以用任何 AI 模型 |
| 📟 **终端界面** | 在终端中使用 |
| 🖥️ **桌面应用** | 也有 macOS 桌面版 |
| 🧩 **插件扩展** | 支持 Agent 切换 |

---

## 📥 安装

```bash
# 安装脚本（最快）
curl -fsSL https://opencode.ai/install.sh | sh

# npm
npm install -g opencode

# Homebrew（macOS/Linux）
brew install opencode

# Go
go install github.com/anomalyco/opencode@latest
```

---

## 🚀 快速开始

```bash
# 启动 OpenCode
opencode

# 进入后直接输入指令
> 帮我写一个 Python 猜数字游戏
```

## 🔌 支持的模型

OpenCode 支持连接 **任何 AI 模型**：

| 模型 | 配置难度 | 说明 |
|------|---------|------|
| 🐋 DeepSeek | ⭐ 简单 | OpenAI 兼容格式 |
| 🧠 GLM | ⭐ 简单 | OpenAI 兼容格式 |
| 🤖 Claude | ⭐⭐ 中等 | 需要 Anthropic API Key |
| 📝 GPT | ⭐ 简单 | OpenAI API Key |
| 🆓 免费模型 | ⭐ 简单 | Gemini/Groq/Ollama |

> 💡 **推荐**：OpenCode + DeepSeek = 免费工具 + 便宜模型！

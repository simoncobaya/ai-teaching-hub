# 📖 OpenCode 基础用法

---

## 启动

```bash
cd your-project
opencode
```

---

## 界面说明

OpenCode 启动后会显示一个终端界面，包含：

- **对话区域**：和 AI 对话的地方
- **文件预览**：预览 AI 要修改的文件
- **Agent 切换**：按 `Tab` 切换不同的 Agent

---

## 常见用法

### 写代码
```
帮我创建一个 Python 的 todo list 程序
```

### 修改代码
```
把 main.py 里的函数加上类型注解
```

### 找 bug
```
运行程序报错了，帮我看看哪里有问题
```

### 解释代码
```
解释一下这个项目的整体结构
```

---

## 快捷键

| 快捷键 | 作用 |
|--------|------|
| `Tab` | 切换 Agent |
| `Enter` | 发送消息 |
| `Ctrl + C` | 取消当前操作 |
| `Ctrl + D` | 退出 OpenCode |

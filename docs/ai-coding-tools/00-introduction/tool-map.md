## 🎓 推荐学习路线

### 路线 A：零基础入门（推荐！）

```
第一步：注册 DeepSeek → 拿到 API Key
第二步：安装 Claude Code → 学会基本操作
第三步：⭐ 配置 Claude Code + DeepSeek → 开始用！
第四步：做第一个实践项目 → 猜数字游戏
第五步：按兴趣探索其他工具
```

### 路线 B：想要少花钱的方案

```
第一步：下载 TRAE → 注册账号（有免费额度）
第二步：用免费额度体验 AI 编程 → 爱上编程！
第三步：预算允许的话 → 升级 TRAE Pro 或转用 DeepSeek + Claude Code
```

### 路线 C：想要最灵活的方案

```
第一步：注册 DeepSeek + 智谱 GLM
第二步：安装 Claude Code + OpenCode
第三步：安装 CC Switch → 统一管理所有配置
第四步：随时切换不同模型和工具
```

## 🤔 该选哪个工具？

### 根据你的情况选择：

| 你的情况 | 推荐工具 | 原因 |
|---------|---------|------|
| 🆓 想完全免费 | OpenCode | 完全免费开源，支持多种 AI 模型 |
| 💰 预算很少 | DeepSeek + Claude Code | DeepSeek 超便宜，按量付费 |
| 🇨🇳 在国内用 | GLM + TRAE 国内版 | 国内服务，速度快 |
| 🎮 想做项目 | Claude Code + Cursor | 功能最强大 |
| 🔧 想折腾 | CC Switch + OpenCode | 最灵活 |

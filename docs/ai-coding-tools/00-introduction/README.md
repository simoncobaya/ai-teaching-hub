# 🌟 什么是 AI 编程工具？

> 想象一下，你有一个超级聪明的编程老师，24 小时陪在你身边，你说什么它就能帮你写什么代码。这就是 **AI 编程工具**！

---

## 🤔 AI 编程工具是什么？

AI 编程工具就是 **用人工智能来帮你写代码的软件**。

你只需要：
1. 用 **自然语言**（就是普通的话）告诉 AI 你想做什么
2. AI 就帮你 **写出代码**
3. 你可以运行代码，看看效果
4. 如果有问题，AI 还帮你 **修改和调试**

举个例子 👇

```
你说："帮我写一个猜数字游戏，电脑随机选一个 1-100 的数字，
     我来猜，每次猜完告诉我是大了还是小了。"

AI 就会帮你写出完整的代码！🎉
```

## 🛠️ AI 编程工具有哪些种类？

AI 编程工具可以分为几大类：

### 📟 终端工具（CLI）
像聊天一样，在黑色的终端窗口里和 AI 对话。

| 工具 | 特点 |
|------|------|
| <img src="../images/icons/claude-color.png" height="20"> **Claude Code** | 最聪明的编程助手，能理解整个项目 |
| <img src="../images/icons/codex-color.png" height="20"> **Codex CLI** | OpenAI 出品，简单好用 |
| <img src="../images/icons/opencode.png" height="20"> **OpenCode** | 完全免费开源，支持各种 AI 模型 |
| <img src="../images/icons/codebuddy-color.png" height="20"> **CodeBuddy Code** | 腾讯出品，支持子代理和 MCP |

### 🖥️ IDE（代码编辑器）
就像一个超级加强版的编辑器，AI 就在里面帮你写代码。

| 工具 | 特点 |
|------|------|
| <img src="../images/icons/cursor.png" height="20"> **Cursor** | 最流行的 AI 编辑器，像 VS Code 的升级版 |
| <img src="../images/icons/trae-color.png" height="20"> **TRAE** | 字节跳动出品，有免费版，付费解锁更多功能 |
| <img src="../images/icons/qoder-color.png" height="20"> **Qoder** | 阿里巴巴出品，国内版和国际版都有 |
| <img src="../images/icons/codebuddy-color.png" height="20"> **CodeBuddy** | 有 IDE 版和命令行版 |

### 🔧 配置管理工具
帮你管理各种 AI 工具的设置。

| 工具 | 特点 |
|------|------|
| **CC Switch** 🔧 | 一个工具管理所有 AI 工具的配置，超方便！ |

### 🧠 AI 模型/大脑
这些是 AI 的"大脑"，提供智能思考的能力。

| 模型 | 厂商 | 特点 |
|------|------|------|
| <img src="../images/icons/deepseek-color.png" height="20"> **DeepSeek** | 深度求索 | 又便宜又好，性价比之王 |
| <img src="../images/icons/chatglm-color.png" height="20"> **GLM** | 智谱 AI | 国内最强，支持中文特别好 |

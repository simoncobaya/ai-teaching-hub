# 🔄 推荐工作流

> 多个 AI 编程工具怎么配合使用？这里有几种推荐的工作流！

---

## 🌟 工作流一：学习入门（低门槛）

适合刚接触 AI 编程的同学。

```
TRAE（有免费额度）
    ↓
写代码、学编程
    ↓
遇到问题 → TRAE Chat 帮你解答
```

**工具**：只需要 TRAE（免费起步）

**流程**：
1. 打开 TRAE
2. 用 SOLO 模式让 AI 帮你写代码
3. 在 IDE 中修改和测试
4. 遇到问题问 TRAE Chat

---

## 🌟 工作流二：日常编程（性价比）

适合有一定基础、经常编程的同学。

```
Claude Code（终端） + DeepSeek（AI 模型）
    ↓
日常编程、写功能、改 bug
    ↓
需要 IDE 体验 → 打开 Cursor 配置同一个 DeepSeek
    ↓
用 CC Switch 管理配置
```

**工具**：Claude Code + DeepSeek + CC Switch

**流程**：
1. 用 CC Switch 配置好 DeepSeek
2. 终端里用 Claude Code 写代码
3. 需要图形界面时打开 Cursor
4. 遇到难题切换到 Claude 或 Reasoner 模型

---

## 🌟 工作流三：全能开发（灵活）

适合想要最灵活配置的同学。

```
CC Switch（统一管理）
    ↓
┌─────────────┬─────────────┐
│ Claude Code  │   Cursor    │
│ （终端任务）  │ （IDE 任务） │
└─────────────┴─────────────┘
    ↓           ↓
DeepSeek（日常） GLM（中文任务）
    ↓
遇到难题 → 切换到 Claude Sonnet
```

**工具**：Claude Code + Cursor + CC Switch + DeepSeek + GLM

**流程**：
1. CC Switch 统一管理所有配置
2. 日常用 DeepSeek（便宜）
3. 中文文档和代码用 GLM（中文好）
4. 复杂问题切换 Claude（最强）
5. 终端用 Claude Code，IDE 用 Cursor

---

## 🌟 工作流四：国内专用

适合在国内使用的同学。

```
TRAE 国内版（有免费额度）
    ↓
日常编程
    ↓
GLM Coding Plan（中文任务）
    ↓
DeepSeek API（备用，便宜）
```

**工具**：TRAE 国内版 + GLM Coding Plan + DeepSeek

**流程**：
1. 日常用 TRAE 国内版（免费起步、网络快）
2. 需要更强能力时，用 GLM Coding Plan
3. DeepSeek 作为备用（便宜）

---

## 💡 多工具配合的技巧

### 技巧 1：一个项目用多个工具

```
项目规划 → TRAE SOLO 模式（快速生成框架）
    ↓
代码开发 → Claude Code + DeepSeek（日常编程）
    ↓
代码审查 → Cursor（IDE 中检查和优化）
    ↓
测试部署 → Claude Code（运行命令）
```

### 技巧 2：根据任务选工具

| 任务 | 推荐工具 | 原因 |
|------|---------|------|
| 快速原型 | TRAE SOLO | AI 自动驾驶，最快 |
| 写新功能 | Claude Code | 自主开发能力强 |
| 代码补全 | Cursor | Tab 补全体验最好 |
| 复杂项目 | Claude Code + Cursor | 终端 + IDE 配合 |
| 学习编程 | TRAE / OpenCode | 免费起步、简单 |

### 技巧 3：用 CC Switch 管理一切

```
CC Switch
    ├── 供应商管理：DeepSeek / GLM / Claude
    ├── 工具管理：Claude Code / Codex / OpenCode
    ├── 模型切换：一键切换
    └── MCP 管理：视觉/搜索/网页/GitHub
```

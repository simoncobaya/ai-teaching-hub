## 💡 省钱技巧

1. **日常用 DeepSeek**，遇到难题再切换到 Claude
2. **用 CC Switch** 一键切换模型，避免忘记切回便宜的
3. **善用 `/compact`** 减少上下文长度，节省 token
4. **清晰的提示词** 减少 AI 需要重试的次数
5. **先用 TRAE 免费版体验**，不够再考虑付费方案

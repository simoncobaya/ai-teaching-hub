# 👯 CodeBuddy（腾讯云代码助手）

> CodeBuddy 是腾讯推出的 AI 编程助手（腾讯云代码助手），基于混元代码大模型，支持 IDE 和 CLI 双版本。

---

## 🤔 CodeBuddy 是什么？

CodeBuddy（腾讯云代码助手）是腾讯推出的 AI 编程助手，提供两种使用方式：

- 🖥️ **IDE 插件版**：集成到 VS Code、JetBrains 等编辑器中
- 📟 **CLI 命令行版**：在终端中使用

而且有 **国内版和国际版**，同一个工具，登录时选择国内账号或国际账号即可。

---

## 🌟 CodeBuddy 的特点

| 特点 | 说明 |
|------|------|
| 🖥️ **IDE 集成** | 支持 VS Code、JetBrains 等主流 IDE |
| 📟 **CLI 版本** | 也可以在终端中使用 |
| 📁 **仓库级理解** | 能理解整个代码仓库 |
| ✏️ **多文件编辑** | 同时修改多个文件 |
| 🇨🇳 **双版本** | 国内版和国际版，登录时选择即可 |

---

## 📥 安装

### IDE 版

1. 打开你的 IDE（VS Code / JetBrains）
2. 在插件市场搜索 **"CodeBuddy"**
3. 点击安装
4. 重启 IDE

### CLI 版

CodeBuddy CLI 是同一个工具，通过不同入口安装，登录时选择国内或国际账号：

| 入口 | 地址 |
|------|------|
| 🇨🇳 **国内入口** | https://www.codebuddy.cn/cli/ |
| 🌍 **国际入口** | https://www.codebuddy.ai/cli |

> 💡 安装后，可以在 CLI 中选择登录国内账号或国际账号。

---

## 🆚 国内版 vs 国际版

| 对比 | 国内版 | 国际版 |
|------|--------|--------|
| 服务区域 | 中国大陆 | 全球 |
| AI 模型 | 国内模型优先 | 国际模型优先 |
| 网络要求 | 国内网络 | 国际网络 |
| 中文支持 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| 注册方式 | 手机号 | 邮箱/Google |

> 💡 **建议**：在国内使用国内版，海外使用国际版。
